# NUMU — System Rules

## General

- prioritize clarity over complexity
- prioritize restraint over decoration
- build reusable, modular components
- every decision must reinforce material perception

---

## Design Language

Visual tone must be:
- calm
- organic
- architectural
- premium
- tactile

Avoid:
- startup UI patterns
- biotech/pharma visuals
- AI/cyberpunk aesthetics
- overdesigned interfaces

---

## Color System

Use:
- sand / beige
- off-white
- muted dark tones

Investor mode:
- deep dark background (black / brown)
- high contrast but soft

Avoid:
- neon
- gradients
- default Tailwind palette

---

## Typography

- serif for headings
- sans-serif for body
- tight tracking for titles
- generous line-height for body
- strong hierarchy

Typography must feel editorial, not digital.

---

## Layout

- large spacing
- strong margins
- minimal UI elements
- one dominant visual per section

The interface should disappear behind the material.

---

## Motion

- only animate transform and opacity
- slow, subtle, breathable
- no aggressive or flashy animation

Motion should feel organic, like a living system.

---

## Interaction

- smooth scrolling
- progressive reveal of content
- minimal interaction points
- no complex UI logic

Focus on perception, not interaction.

---

## 3D Usage

- use only in hero section
- single central object (floating panel)
- slow rotation only
- no complex shaders or effects

3D must feel calm and physical, not technical.

---

## References

References (e.g. sutera.ch) are used for:
- structure
- layout logic
- interaction rhythm

Never copy:
- visual style
- colors
- typography
- UI components

Always reinterpret into:
- biological
- material
- architectural language

---

## Content Integrity

Do not:
- invent fake data
- fabricate projects
- exaggerate performance

Use only validated content.

---

## Output Quality

The result must:
- impress architects
- attract premium clients
- be credible for investors

Always favor:
- restraint over excess
- clarity over decoration
- system over randomness

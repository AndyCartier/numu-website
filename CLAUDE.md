# CLAUDE.md — NUMU Website Rules

## Core Principle
This is not a generic website.
This is a design-led, material-driven digital experience.

Prioritize:
- clarity
- elegance
- restraint
- architectural quality

Avoid:
- generic startup UI
- AI/cyberpunk aesthetics
- overdesigned interactions

---

## Reference Usage

Reference materials (e.g. sutera.ch) are for:
- structural inspiration
- interaction patterns
- layout logic

DO NOT:
- copy visual style
- copy colors or typography
- replicate UI directly

Always reinterpret into NUMU’s identity:
- biological
- material
- architectural

---

## Architecture

Use:
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- React Three Fiber (only where necessary)

Build:
- reusable components
- scalable structure
- clean separation of concerns

---

## Core Feature

Implement a dual-mode system:

1. Visitor mode
2. Investor mode

Switch must:
- change navigation
- change content hierarchy
- change CTAs
- feel like a transformation, not a toggle

---

## Visual Direction

Design must feel:
- premium
- tactile
- calm
- organic
- architectural

Color palette:
- sand / beige
- off-white
- muted dark tones

Avoid:
- neon
- default Tailwind colors
- SaaS gradients

---

## Typography

- Pair serif (headings) + sans-serif (body)
- Tight tracking for headings
- Generous line-height for body
- Strong hierarchy

---

## Motion

- Only animate transform and opacity
- Motion should feel slow, organic, breathable
- Avoid aggressive or flashy animation

---

## Components

Every component must have:
- hover state
- focus state
- active state

Spacing must be:
- consistent
- intentional
- system-based

---

## Images

- Prioritize real assets over placeholders
- Use texture and material detail
- Maintain strong visual hierarchy

---

## Content Integrity

Do not:
- invent fake data
- create fake projects
- fabricate metrics

Use provided content as source of truth.

---

## Output Quality

The result must:
- impress architects
- attract premium clients
- be credible for investors

Favor:
- restraint over excess
- clarity over decoration
- system over randomness
